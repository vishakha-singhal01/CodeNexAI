# CodeNexAI Document your code

**Select code in your editor to generate documentation using CodenexAI.**

This extension helps you quickly generate documentation for your code selections using the power of CodenexAI.

## Features

*   Right-click on a code selection in the editor.
*   Choose "CodenexAI: Generate Documentation for Selection" from the context menu.
*   Helps you in documenting your code so that you can understand it after 6 months , 12 months or an eternity later (xd)

## Requirements

*  You have to first signup at codenexai.com & verify your mail to use this extension.

## How to Use

1.  Install the extension.
2.  Open a code file.
3.  Select the code snippet you want to document.
4.  Right-click and select "CodenexAI: Generate Documentation for Selection".

## Extension Settings

This extension does not contribute any VS Code settings at the moment.
